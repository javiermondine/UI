# Dynamic User Interface Interactions

Proyecto de práctica de **The Odin Project** - Lección "Dynamic User Interface Interactions"

## Contenido

Este proyecto implementa dos componentes interactivos con JavaScript vanilla:

### 1. Dropdown Menu
- Menú desplegable que muestra/oculta opciones al hacer clic
- Código reutilizable para múltiples dropdowns en la misma página
- Cierre automático al hacer clic fuera del menú
- Soporte para tecla Escape

### 2. Image Carousel
- Carrusel de imágenes con navegación
- Botones de anterior/siguiente
- Avance automático cada 5 segundos
- Pausa al pasar el mouse
- Indicadores clicables para navegación directa

## Tecnologías

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript Vanilla (ES6+)

## Estructura

```
dynamic-ui/
├── index.html      # Estructura HTML
├── styles.css      # Estilos
├── dropdown.js     # Lógica del dropdown
├── carousel.js     # Lógica del carrusel
└── README.md       # Este archivo
```

## Características

- **Código modular**: Cada componente en su propio archivo
- **Reutilizable**: Los componentes funcionan con múltiples instancias
- **Responsive**: Adaptable a diferentes tamaños de pantalla
- **Accesible**: Soporte para teclado y buenas prácticas
